# case-extraction-tools
The Case Extraction Tools repository contains software utilities and data pipelines for converting research publications, project reports, and educational documentation into structured case entries for the AAB AI Education Case Registry.

The goal of this project is to support scalable collection of global AI education practices by transforming unstructured documents into standardized case records.

Tools in this repository may include:

PDF and document parsing tools

AI-assisted research summarization tools

metadata extraction scripts

research-paper-to-case conversion pipelines

dataset ingestion and validation scripts

These tools help reduce the manual effort required to build a comprehensive registry of AI education implementations and evidence.

The project welcomes contributions from developers, data scientists, and education researchers interested in building open infrastructure for AI education research.
